## 📦 프로젝트 개요
이 프로젝트는 **Vite + Vue + TypeScript**를 이용해 제작한 **한가위(추석) 테마 웹페이지**입니다.  
보름달 아래 토끼가 복을 빚고, 복주머니를 클릭하면 랜덤한 덕담이 등장하는 인터랙티브 페이지입니다.  
별빛 배경과 애니메이션을 활용해 한가위의 따뜻한 분위기를 표현했습니다.
